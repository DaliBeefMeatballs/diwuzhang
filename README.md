# 🎨 互动数据可视化展示

基于第5章matplotlib示例的Web交互版本，使用Chart.js实现的数据可视化项目。

## 📊 项目简介

这个项目将Python matplotlib的数据可视化示例转换为交互式Web应用，展示了多种图表类型和数据可视化技术。项目包含5个主要的数据分析模块，每个模块都具有丰富的交互功能。

## ✨ 功能特性

### 🏭 产品销售额分析
- 支持折线图、柱状图、饼图三种显示方式
- 动态切换图表类型
- 动画效果展示
- 显示12个月的产品A与产品B销售数据对比

### 🐕🐱 养猫养狗人群比例分析
- 展示12个国家的养猫与养狗人群比例
- 支持按养猫比例、养狗比例或默认排序
- 柱状图与折线图切换显示
- 水平条形图展示，便于比较

### 📱 抖音用户增长分析
- 2017-2018年抖音用户增长倍数展示
- 支持2017年、2018年用户地区分布的饼图展示
- 对比分析功能，同时显示两年数据
- 突出显示四线及以外城市的显著增长

### 🌡️ 气温与降水量关系分析
- 组合图表展示气温、降水量、蒸发量关系
- 双Y轴设计，左侧显示水量数据，右侧显示气温
- 支持显示全部数据、仅气温、仅水量
- 夏季数据高亮功能

### 🚗 汽车销售情况分析
- 2018年上半年月度销售额柱状图
- 地区销量折线图展示
- 堆叠分析图表
- 动画展示功能

## 🛠️ 技术栈

- **前端框架**: 纯HTML5 + CSS3 + JavaScript
- **图表库**: Chart.js 3.x
- **样式**: CSS Grid + Flexbox 响应式布局
- **交互**: 原生JavaScript事件处理
- **动画**: CSS3动画 + Chart.js动画

## 🚀 使用方法

### 本地运行
1. 克隆或下载项目文件
2. 直接在浏览器中打开 `index.html` 文件
3. 或使用本地服务器：
   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 使用Node.js
   npx serve .
   ```

### 在线部署
项目可部署到任何静态网站托管服务：
- GitHub Pages
- Netlify
- Vercel
- 等等

## 🎯 交互功能

每个图表模块都包含以下交互功能：

1. **图表类型切换**: 点击按钮切换不同的图表展示方式
2. **数据排序**: 支持不同维度的数据排序
3. **动画效果**: 平滑的加载和交互动画
4. **响应式设计**: 自适应不同屏幕尺寸
5. **悬停提示**: 鼠标悬停显示详细数据信息

## 📱 响应式设计

项目采用响应式设计，支持：
- 桌面端：多列网格布局
- 平板端：自适应布局
- 移动端：单列堆叠布局

## 🎨 设计特色

- **现代UI设计**: 渐变色背景、圆角卡片、阴影效果
- **统一配色**: 紫蓝渐变色主题
- **微交互**: 悬停效果、按钮动画、加载动画
- **数据可视化**: 清晰的图例、坐标轴标签、数据提示

## 📊 数据来源

项目数据来源于matplotlib第5章示例，包括：
- 产品销售数据：12个月的销售额对比
- 宠物饲养数据：12个国家的养猫养狗比例
- 抖音用户数据：2017-2018年用户增长和地区分布
- 气象数据：全年气温、降水量、蒸发量
- 汽车销售数据：月度销售额和地区销量
